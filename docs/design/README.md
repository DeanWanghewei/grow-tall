# 设计草稿(UI Mockups)

脑暴阶段产出的 HTML 真机草图,用浏览器直接打开即可查看(每个都带真实 CSS + 动效)。这是设计参考,非最终实现代码。

## 阅读顺序

| 文件 | 内容 | 备注 |
|------|------|------|
| `01-visual-style.html` | 三套卡通主题方向(暖阳长颈鹿 / 清新薄荷 / 彩虹活力) | 最终决定:三套都要 → 多主题系统 |
| `02-app-map.html` | 整体页面结构(4 标签 + 凸起「记一笔」) | 信息架构定稿 |
| `03-home.html` | 首页「成长墙」+「记一笔」底部弹层 | 卡通化重做版 |
| `04-safe-area.html` | iOS(灵动岛)+ HarmonyOS 6(挖孔)安全区适配 | 上下安全区、凸起按钮避让 |
| `05-curves-album.html` | 成长曲线(百分位带)+ 时光相册 + 幻灯片v1 | 曲线为旗舰页 |
| `06-slideshow.html` | 成长幻灯片两个方向 | 最终选定 **方向 A:成长墙时间轴** |
| `07-settings-theme.html` | 我的(设置)+ 主题选择器 + 早期 S3 表单 | S3 部分已被 `08` 取代 |
| `08-storage.html` | 图片存储极简 UI(可用/不可用) | S3 最终设计:部署时检测,UI 只露状态 |

> 设计规格见 `docs/superpowers/specs/2026-06-21-kids-growth-tracker-design.md`。
